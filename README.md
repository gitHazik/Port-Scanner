# Python Port Scanner

A simple Python-based port scanner that scans a specified range of ports on a target IP address. It identifies open ports by attempting TCP connections and reports the results.

This tool is useful for:
- Network troubleshooting
- Testing firewall configurations
- Identifying open ports on a machine

## Features
- Scans a range of TCP ports on a specified IP address
- Reports which ports are open
- Lightweight and easy to use
- Customizable port range

## Requirements
- Python 3.x
